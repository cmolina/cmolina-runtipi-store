# Shelfmark

Shelfmark is a self-hosted web interface for searching and downloading books and audiobooks from multiple sources. It acts as a unified search hub rather than a library manager.

## Features

- **Multi-source search**: Aggregates torrent, usenet, IRC, and direct download sources in one interface
- **Metadata enrichment**: Integrates with Hardcover, Open Library, and Google Books for rich book details
- **Audiobook support**: Dedicated audiobook search and processing capabilities
- **Multi-user workflows**: Request approval system with user management and authentication options
- **Real-time progress**: WebSocket-based download queue with live status tracking
- **Download client integrations**: Works with qBittorrent, Transmission, Deluge, SABnzbd, and NZBGet
- **Library integrations**: Automatic imports to Calibre, Calibre-Web, Grimmory, and Audiobookshelf
- **Flexible search modes**: Direct (query sources) or Universal (metadata-first) search

Shelfmark is focused on search and download only — it does not handle library management or background automation.
